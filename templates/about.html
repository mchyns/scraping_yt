Tentang Sistem

Sistem Analisis Sentimen Komentar YouTube

Sistem ini dibuat untuk BPS (Badan Pusat Statistik) untuk melakukan analisis sentimen pada komentar YouTube.

Metode Analisis:

  • Lexicon-Based: Menggunakan kamus kata positif/negatif
  • VADER: Valence Aware Dictionary for Sentiment Reasoning
  • TextBlob: Library NLP untuk analisis sentimen
  • Rule-Based: Kombinasi metode dengan aturan kustom

Fitur:

  • Scraping komentar YouTube dengan API
  • Analisis sentimen dengan 4 metode
  • Perbandingan hasil metode
  • Export data ke Excel
